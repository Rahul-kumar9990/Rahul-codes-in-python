#Python Problem Solving & Assignments

This repository serves as a personal log of my journey learning Python. Rather than simple notes, each file represents a **coding assignment or problem-solving challenge** that I completed to master core Python concepts. 

---

## 🛠 My "Learning-by-Doing" Approach
I believe the best way to learn Python is by writing code, breaking it, and fixing it. This repository contains the assignments and problems I tackled to move from basic syntax to intermediate data manipulation.

| Topic | Focus Area | What I Solved |
| :--- | :--- | :--- |
| **Fundamentals** | Syntax & Logic | Basic I/O and script execution challenges. |
| **Operators** | Mathematical Logic | Solving arithmetic and relational logic problems. |
| **Strings** | Text Manipulation | Challenges involving slicing, formatting, and string parsing. |
| **Conditionals** | Decision Making | Logic puzzles using `if/else` branching. |
| **Loops** | Iteration | Challenges to automate repetitive tasks. |
| **List Comprehensions**| Optimized Coding | Advanced challenges to write concise, Pythonic code. |
| **Dictionaries** | Structured Data | Assignments on mapping and efficient data lookups. |

## 🚀 Why this repo?
* **Hands-on Proof:** Each file is a record of a problem I solved. 
* **Growth Progression:** The repo reflects a clear path from simple logic to more complex data structures.
* **Practical Foundation:** This is the exact coding foundation I used to bridge into **Data Analytics (NumPy/Pandas).**

## 💡 How I Use These Files
Each file documents my solution to a specific learning milestone. 
1. **The Problem:** I identified a concept (e.g., Loops).
2. **The Assignment:** I wrote code to solve a specific challenge related to that concept.
3. **The Result:** The code successfully runs and solves the problem, serving as my reference point for future projects.

---

## 📈 My Learning Path
1. **Master the Basics:** Start with fundamentals and operators.
2. **Handle Logic:** Use conditionals and loops to control program flow.
3. **Data Handling:** Master strings, lists, and dictionaries to manage data—the essential precursors to working with **DataFrames**.
